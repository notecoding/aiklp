#인공지능 프로젝트
박도현 김용효 이근석

### 1. 프론트엔드 설치

```bash
# 프로젝트 클론
git clone <repository-url>
cd front

# 의존성 설치
npm install

# 개발 서버 실행
npm start

```

서버 실행 후: http://localhost:3000

### 2. 백엔드 설치 및 실행

```bash
# 백엔드 디렉토리로 이동
cd backend

# 필요한 패키지 설치

pip install -r requirements.txt

# Flask 서버 실행
python app.py

```

서버 실행 후: http://localhost:5000
